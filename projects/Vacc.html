Vaccination Date Control Program - C++ with Qt

  • Tech Stack: C++ Programming Language, QT Framework for GUI Developpement, SQlLite for Data Storage
  • Rapport URL: Contact Me

This project involves developing a program in C++ using the Qt framework to manage and control vaccination dates. The main objective is to create a reliable and user-friendly application that helps users track vaccination schedules, set reminders, and maintain vaccination records.

Key Features:

  • Vaccination Schedule Management: Allows users to input and manage vaccination dates for different individuals or groups.
  • Reminder System: Implements a notification system to alert users of upcoming vaccination dates.
  • Data Storage and Retrieval: Maintains a database of vaccination records, providing easy access and management of historical data.
  • User Interface: Develops a clean and intuitive GUI (Graphical User Interface) using Qt, enhancing user experience and interaction.
  • Data Reporting: Generates reports and summaries of vaccination schedules and adherence.
